#  Airbnb Analytics Dashboard

**Tool Used:** Power BI, DAX, Power Query  
**Type:** Data Analytics | Revenue & Booking Analysis

## Objective
Track bookings, revenue, occupancy rates, and guest ratings 
to identify seasonal trends and top-performing properties.

##  Key KPIs Tracked
- Monthly Bookings & Revenue Growth
- Property Occupancy Rate
- Guest Ratings & Reviews
- Top Performing Properties

##  Key Insights
- Identified seasonal booking trends
- Highlighted top revenue-generating properties
- Analyzed guest satisfaction patterns

##  Skills Demonstrated
- Power BI Dashboard Design
- DAX & Power Query
- Revenue Analytics
- Trend Analysis
